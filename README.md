# 프로젝트 개요

### <span style="color: green">레이서들을 위한 스켈레톤 코드를 제공하기 위해 엘리서가 작성한 간단한 레시피 공유 앱 입니다.</span>

## 주요 기능

- 회원가입, 로그인
- 글 작성, 삭제, 수정, 조회
- 좋아요, 댓글, 스크랩
- 프로필 조회, 수정
- 레시피 검색

## 사용된 기술 스택

- node.js
- react

### 인증 방식

- jwt

### 서버
- express

### DB
- mongoDB


## 프로젝트 실행 방법

### 클라이언트
```bash
npm i # 패키지 설치
npm start # react 앱 실행
```

### 서버
```bash
cd backend # backend 폴더로 이동
npm i # 패키지 설치
node index # express 서버 실행
```
